# Dashboard de Finanzas Personales

Dashboard autónomo construido con React, Tailwind CSS, Recharts y Supabase.

## Características
- Visualización de ingresos, gastos y ahorro acumulado.
- Gráficos interactivos de evolución anual y gastos por categoría.
- Gestión de transacciones con persistencia en Supabase.
- Interfaz en castellano con moneda en Euros.

## Instalación
1. `npm install`
2. Configura tu `.env.local` con las credenciales de Supabase.
3. `npm run dev`
